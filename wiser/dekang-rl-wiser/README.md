# 多智能体联合学习
1. 每个智能体分配1w元
2. 每个智能体只能操作自己的1w元
3. 其他智能体上一步的操作和收益也会作为观察值
4. （可选）其他智能体的操作不当会影响自己的加分，这样智能体就具备了团队精神（猜测）
